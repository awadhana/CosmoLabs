# CosmoLabs

Marketing website for **CosmoLabs** — a multidisciplinary consulting firm providing professional services in:

- **Computer Science & Software** — software engineering, cloud & DevOps, AI/ML systems, cybersecurity, data engineering
- **Aerospace Engineering** — mission & systems engineering, GNC, propulsion & structures analysis, avionics & flight software
- **Nuclear Engineering** — reactor safety analysis, NRC licensing support, thermal-hydraulics, radiation shielding, decommissioning

Trusted by leading firms including Amazon, The Home Depot, Microsoft, Delta Air Lines, and more.

## Stack

Single self-contained static page (`index.html`) — no build step, no framework. Inline CSS/JS, Google Fonts, inline SVG icons.

## Run locally

Open `index.html` in a browser, or serve it:

```sh
npx serve .
```

## Deploy

Deployed on [Vercel](https://vercel.com). Any push to `main` can be redeployed with:

```sh
vercel --prod
```

## Notes

Contact details on the site (email, phone, address) are placeholders — update them in `index.html` before going live with real contact channels.
